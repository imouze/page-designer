属性
属性由层触发，激活层即可触发，添加层时是否要同步创建
属性的生存周期在于层是否激活，没激活就不显示，所以可以同步创建属性
支持拖动
支持展开收缩
只要层是激活的，属性不会消失


数据
属性会更新数据
拖动图层会更新位置数据
导入图片或上传图片会更新地址数据以及大小数据
删除会移除数据
